# 🌍 EcoQuest: Gamified Eco-Education Platform

EcoQuest is a state-of-the-art, gamified learning platform designed to bridge the gap between environmental awareness and real-world action. By transforming complex climate concepts into interactive missions and rewards, EcoQuest empowers students to become proactive guardians of our planet.

✨ Key Features

🎮 Gamified Missions
- Ocean Cleanup: Pilot advanced vessels to neutralize plastic contamination.
- Pollution Patrol: Scramble drones to monitor and reduce industrial emissions.
- Tree Tycoon: Build and manage your own carbon-capturing forest.
- Grid Guardian: Master the balance of renewable energy distribution.

📚 Nature Library
- A cinematically curated archive of environmental wisdom.
- Deep dives into Oceanography, Terrestrial Ecosystems, and Renewable Energy.
- Direct links to expert resources and books for extended learning.

🏆 Achievement System
- Eco Assets: Earn points for completing missions and quizzes.
- Day Streaks: Build consistency through daily ecological engagement.
- Hero Leaderboard: Compete with students globally to climb the ranks of Eco Guardians.

📸 Green Stories
- Share real-world conservation efforts with the community.
- "Respect" system to encourage and validate positive environmental impact.

🎨 Design Philosophy
EcoQuest features a Premium Modern UI designed to wow users:
- Glassmorphism: Sleek, semi-transparent interfaces that feel light and futuristic.
- Cinematic Backgrounds: High-definition video backgrounds for an immersive "Nature-First" experience.
- Dynamic Animations: Smooth transitions and interactive elements that make the platform feel alive.

🛠️ Technology Stack
- Frontend: Vanilla HTML5, CSS3 (Custom Glassmorphism Design System), JavaScript (ES6+).
- Backend: Node.js & Express (Robust API Architecture).
- Database: Supabase (Real-time data, Authentication, and Scalable Storage).
- Animation: CSS Keyframes & Backdrop Filters.

🚀 Getting Started

Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- [Supabase Account](https://supabase.com/) (for database features)

Installation
1.Clone the repository:
   ```bash
   git clone c
   ```

2.Setup Backend:
   - Navigate to the `backend` folder.
   - Create a `.env` file and add your Supabase credentials:
     ```env
     SUPABASE_URL=your_url
     SUPABASE_SERVICE_ROLE_KEY=your_key
     PORT=4000
     ```
   - Install dependencies and start the server:
     ```bash
     npm install
     npm start
     ```
